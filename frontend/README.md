# Intelligent Tourism Frontend

This frontend is built with React, Vite, React Router, and Tailwind CSS.

## Local Run

From the repository root:

```powershell
cd frontend
npm install
npm run dev
```

The app runs on `http://localhost:5173`.

## Build

```powershell
npm run build
npm run preview
```

## Frontend Areas

- Home and attraction discovery
- Destination details and reviews
- Recommendations
- Map view
- Itinerary builder
- Booking center
- Notifications
- Profile and authentication
- Community page
- Analytics dashboard

## Environment

Set `VITE_API_BASE` if the backend is not running at `http://localhost:8080/api`.
